Additional Resources

Links

Here are some links you can use to learn more fundamentals about NLP, build NLP applications, and play around with applications that use NLP.

  • Natural Language Toolkit - An easy-to-use platform to build programs that work with language data. It has many interfaces for developers to use and you can learn about all of it using the documentation.
  • Write With Transformer - A modern neural network that allows you to write words and get suggestions to complete your sentences. This platform resembles modern NLP applications.
  • NLP Module - Dr. Greg Durrett's NLP module where you can watch videos and view slides that will greatly increase your knowledge of NLP.
  • Open AI API - This website shows a wide range of examples of common tasks that NLP applications can do.
  • Hugging Face - A platform that provides open-source interfaces that use language processing. You can build NLP applications using this platform.

Ideas for Projects

  • Build a simple sentence completer app that uses the previous words in order to suggest the next word. The second link on the first section of this page above is a great example of a sentence completer app. However, that app is very complicated to make, so your app can be much more simple. You can use probabilities to suggest the next word based on the sentence starters. You can use this resource to get started.
  • Build a sentiment analysis app. Sentiment analysis is identifying the tone or emotions of any text. You can use this resource to get started.
